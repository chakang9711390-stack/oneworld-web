# 一世界｜V1 前后端研发拆解 v1

## 一、前端研发拆解

### 1. 基础框架
- 项目初始化
- 路由系统
- 全局样式变量
- 深浅主题切换
- 响应式布局

### 2. 公共组件
- 顶部导航组件
- 按钮组件
- 卡片组件
- 搜索框组件
- 弹窗组件
- 面包屑组件
- 指标卡片组件

### 3. 页面开发
- 首页
- OPC 行业库页
- OPC 项目卡片页
- 工作流行业库页
- 工作流职位页
- 工作流场景页
- 我的工作台页

### 4. 交互逻辑
- 页面切换
- 卡片点击进入
- 面包屑返回
- 登录/注册弹窗逻辑
- tab 切换

## 二、后端研发拆解

### 1. 基础能力
- 用户体系基础表
- 行业表
- OPC 模板表
- 工作流职位表
- 工作流场景表
- 工作台项目表

### 2. 接口层
建议首批提供以下接口：

- 用户注册
- 用户登录
- 获取 OPC 行业列表
- 获取某行业下 OPC 项目卡片列表
- 获取工作流行业列表
- 获取某行业下职位列表
- 获取某职位下工作场景列表
- 获取当前用户工作台概览数据

### 3. 暂可后置的接口
- 创建我的 OPC
- 生成我的工作流
- 搜索接口
- 用户资产编辑接口

---

## 三、联调建议

1. 前端可先用静态 mock 数据完成页面结构
2. 后端优先补基础查询接口
3. 登录注册若暂不联通，可先用伪状态完成演示
4. 生成型能力统一在下一阶段接入
