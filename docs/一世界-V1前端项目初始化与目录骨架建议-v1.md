# 一世界｜V1 前端项目初始化与目录骨架建议 v1

## 1. 推荐初始化命令

建议使用 Next.js + TypeScript 初始化：

```bash
npx create-next-app@latest oneworld-web --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
```
