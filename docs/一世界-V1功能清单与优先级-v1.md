# 一世界｜V1 功能清单与优先级 v1

## 一、P0 必须完成

### 1. 平台框架
- 顶部导航切换
- 页面路由切换
- 通用卡片组件
- 通用按钮样式
- 深浅主题切换
- 响应式适配

### 2. 首页
- Hero 展示
- 搜索输入框展示
- 开始探索按钮展示
- 三入口卡片跳转

### 3. OPC 模块
- 行业卡片浏览
- 行业进入详情页
- 行业内 OPC 项目卡片展示
- 面包屑返回

### 4. 工作流模块
- 行业卡片浏览
- 职位卡片浏览
- 工作场景卡片浏览
- 面包屑返回

### 5. 工作台模块
- 工作台页面结构
- 新建项目按钮占位
- 指标卡片展示

### 6. 登录注册模块
- 登录弹窗打开/关闭
- 注册弹窗打开/关闭
- 登录/注册 tab 切换
- 基础表单布局

---

## 二、P1 建议尽快补齐

- 搜索框真实搜索逻辑
- 登录/注册表单校验
- 未登录与已登录状态区分
- 用户工作台空状态设计
- 创建我的 OPC 的点击后占位流程
- 生成我的工作流的点击后占位流程

## 三、P2 后续扩展

- AI 推荐入口
- OPC 自动生成逻辑
- 工作流自动生成逻辑
- 用户资产编辑能力
- 用户生成记录历史
- 收藏、继续使用、复制等能力
- 运营后台与内容管理能力
