# 一世界｜V1 后端起步层说明 v1

本轮已经在 `oneworld-web` 项目中补入首批后端起步层，用于让项目从纯前端骨架进入“前端 + API + Prisma 草案”阶段。

## 已补内容

### 1. Prisma 目录
- `oneworld-web/prisma/schema.prisma`
- `oneworld-web/prisma/seed.ts`

### 2. API 路由草案
- `GET /api/opc/industries`
- `GET /api/opc/industries/:industrySlug/projects`
- `GET /api/workflow/industries`
- `GET /api/workflow/industries/:industrySlug/roles`
- `GET /api/workflow/roles/:roleSlug/scenes`
- `GET /api/workspace/overview`
- `GET /api/auth/me`

### 3. 服务端 mock 数据层
- `src/server/api-data.ts`

### 4. 环境变量模板
- `.env.example`

## 当前定位

当前后端层还属于 **V1 起步骨架**，重点是：
- 先把 API 路径与返回结构定下来
- 让前端未来从 mock 数据切到 API 更顺
- 为接真实 Prisma / PostgreSQL 做准备

## 下一步建议

1. 安装 Prisma 相关依赖
2. 接 PostgreSQL
3. 生成首个 migration
4. 将 API 路由从本地 mock 切到 Prisma 查询
