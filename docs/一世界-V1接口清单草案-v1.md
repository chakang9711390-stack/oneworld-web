# 一世界｜V1 接口清单草案 v1

## 1. 账号体系

### POST /api/auth/register
用于邮箱注册。

### POST /api/auth/login
用于邮箱登录。

### GET /api/auth/me
用于获取当前登录用户信息。

---

## 2. OPC 模块

### GET /api/opc/industries
获取 OPC 行业列表。

### GET /api/opc/industries/:industryId/projects
获取某行业下的 OPC 项目卡片列表。

---

## 3. 工作流模块

### GET /api/workflow/industries
获取工作流行业列表。

### GET /api/workflow/industries/:industryId/roles
获取某行业下职位列表。

### GET /api/workflow/roles/:roleId/scenes
获取某职位下工作场景列表。

---

## 4. 工作台模块

### GET /api/workspace/overview
获取当前用户工作台概览数据。

### GET /api/workspace/projects
获取当前用户项目列表。

---

## 5. 后续预留接口

### POST /api/opc/create
创建我的 OPC。

### POST /api/workflow/generate
生成我的工作流。
