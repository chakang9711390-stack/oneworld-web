# 一世界｜V1 数据结构草案 v1

## 1. User 用户
用于表示平台用户。

建议字段：
- id
- email
- password_hash（若自建账号体系）
- nickname
- avatar
- auth_provider
- created_at
- updated_at

---

## 2. Industry 行业
用于承接 OPC 与工作流的行业分类。

建议字段：
- id
- name
- type（opc / workflow / both）
- description
- sort_order
- status

---

## 3. OpcProjectTemplate OPC 项目模板
用于展示某行业下的 OPC 项目模板或案例卡片。

建议字段：
- id
- industry_id
- title
- description
- is_create_entry
- status
- sort_order

---

## 4. WorkflowRole 工作流职位
用于表示某行业下的职位卡片。

建议字段：
- id
- industry_id
- name
- description
- sort_order
- status

---

## 5. WorkflowScene 工作流场景
用于表示某职位下的工作场景卡片。

建议字段：
- id
- role_id
- title
- description
- is_generate_entry
- status
- sort_order

---

## 6. WorkspaceProject 工作台项目
用于承接用户在工作台中的项目资产。

建议字段：
- id
- user_id
- title
- project_type
- source_type
- status
- summary
- created_at
- updated_at

---

## 7. WorkspaceMetric 工作台指标
当前原型中的数字卡片可由聚合查询得到，不一定需要独立存表。

建议通过以下统计聚合得到：
- 当前 OPC 数
- 当前工作流数
- 当前项目数
