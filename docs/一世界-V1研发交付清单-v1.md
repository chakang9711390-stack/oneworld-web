# 一世界｜V1 研发交付清单 v1

## 一、产品侧交付
- V1 产品研发总说明
- 页面清单与页面说明
- 功能清单与优先级
- 数据结构草案
- 接口清单草案
- 前后端研发拆解

## 二、设计侧交付
- 最新高保真 UI 原型 HTML
- 如有切图 / 标注 / 设计规范，后续补充进入 assets 目录

## 三、研发侧首轮目标
- 完成平台框架搭建
- 完成首页与主路径页面
- 完成 OPC / 工作流 / 工作台结构页
- 完成登录注册弹窗框架
- 完成静态数据或 mock 数据演示

## 四、当前进入研发的判断标准
满足以下条件即可进入首轮研发：

1. 结构已经稳定
2. 页面边界已经清楚
3. 模块关系已定义
4. 必做与占位内容已区分
5. 前后端拆解已可执行
